# Zepto Inventory, Pricing & Revenue Analysis using PostgreSQL

## Project Overview
This project focuses on analyzing Zepto retail product data using PostgreSQL. The goal was to perform inventory analysis, pricing optimization, discount analysis, revenue estimation, and stock availability evaluation using SQL.

## Tools Used
- PostgreSQL
- pgAdmin 4
- SQL
- GitHub

## Key Analysis Performed
- Data cleaning and preprocessing
- Null value detection
- Product category analysis
- Stock availability analysis
- Revenue estimation by category
- Top discount product analysis
- Price per gram calculation
- Inventory weight analysis
- Product grouping using CASE statements

## SQL Concepts Used
- SELECT
- GROUP BY
- ORDER BY
- HAVING
- CASE
- AGGREGATE FUNCTIONS
- DATA CLEANING
- UPDATE
- DELETE
